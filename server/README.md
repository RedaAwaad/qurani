# Qurani Back-end

### Run Locally

```bash
  git clone https://github.com/RedaAwaad/qurani.git
```

Go to the project directory

```bash
  cd qurani/server
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Now, you can open the app through the following URL [http://localhost:8000](http://localhost:8000)
